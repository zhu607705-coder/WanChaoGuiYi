# WanChaoGuiYi

Unity project workspace for 《万朝归一：九州帝业》.

## Current Stage

- Stage: MVP initialization.
- Target: Domestic China-focused playable Demo.
- Engine: Unity.
- Language: C#.
- Data: JSON tables under `Assets/Data`.

## First Playable Slice

The first playable slice should prove:

1. A node map can be clicked and inspected.
2. Regions have ownership, resources, land structure and neighbors.
3. A turn can resolve economy and risk changes.
4. Emperor mechanics change outcomes.

## Unity Import Notes

Open this directory as the Unity project root.

The placeholder editor baseline is Unity `2022.3.x LTS`. Week 1 should lock the exact installed editor version and let Unity regenerate complete `ProjectSettings` metadata.
