# 《万朝归一：九州帝业》世界观设定

## 一、核心命题

**天命无常，惟德是辅。**

这不是一个"谁的疆域大谁赢"的游戏。这是一个关于**王朝为何兴衰**的模拟器。玩家扮演的不是地图涂色者，而是天命的承载者——你的每一个决策都在回答一个问题：**这个王朝是否值得延续？**

> "皇天无亲，惟德是辅。" ——《尚书·蔡仲之命》

## 二、时代跨度

### 游戏覆盖的历史纵深

| 时代 | 对应历史时期 | 核心矛盾 | 代表帝皇 |
|------|-------------|----------|----------|
| **古典期** (classical) | 秦→东汉 (前221-220) | 统一与治理的张力 | 秦始皇、刘邦、汉武帝 |
| **中古期** (medieval) | 三国→五代 (220-960) | 分裂与重建的循环 | 曹操、李世民、赵匡胤 |
| **近世期** (early_modern) | 宋→清 (960-1912) | 制度成熟与帝国极限 | 朱元璋、康熙 |

### 为什么是这三个时代？

这三段历史覆盖了中国帝制的核心命题：

1. **古典期**：如何从分裂走向统一？统一后如何治理？
2. **中古期**：门阀、军阀、外族——统一为什么反复崩溃？
3. **近世期**：科举、文官、火器——制度能解决所有问题吗？

> 参考：渤海小吏《三国争霸》《两晋悲歌》《南北归一》《大唐气象》《唐宋之变》五部著作的分期逻辑。

## 三、帝皇选择逻辑

### 13 位帝皇的选择标准

不是选"最伟大"的皇帝，而是选**最具机制差异化潜力**的帝皇。

| 选择维度 | 标准 | 说明 |
|----------|------|------|
| **机制独特性** | 每位帝皇必须有不可替代的独特机制 | 不能只做数值差异 |
| **历史代表性** | 覆盖不同历史时期和治理模式 | 从法家到儒家到多元治理 |
| **矛盾张力** | 每位帝皇都有明确的"强项"和"历史代价" | 没有完美的帝皇 |
| **AI 差异** | 每位帝皇的 AI 行为风格可区分 | 扩张型、治理型、保守型 |

### 13 位帝皇及其核心机制

| 帝皇 | 机制名称 | 核心矛盾 | 史料来源 |
|------|----------|----------|----------|

## 四、九州概念

### "九州"在游戏中的定义

游戏中的"九州"不是《禹贡》的九个州，而是**华夏疆域的文化符号**。

> "九州"从地理区划逐渐演变为文化认同符号，成为中华民族"大一统"观念的核心表达。——基于《尚书·禹贡》《周礼·职方氏》《尔雅·释地》综合

### 九州的地理范围

游戏覆盖的 56 个区域，参照以下历史地理文献：

| 参照文献 | 作用 |
|----------|------|
| 《汉书·地理志》 | 西汉行政区划，古典期区域基础 |
| 《隋书·地理志》 | 隋唐行政区划，中古期区域基础 |
| 《宋史·地理志》 | 宋代行政区划，近世期区域基础 |
| 《读史方舆纪要》 | 历代军事地理，区域战略价值参考 |
| 渤海小吏五部著作 | 具体战役地理、区域风俗描述 |

### 区域分类

| 区域类型 | 示例 | 核心特征 |
|----------|------|----------|
| **帝都核心区** | 长安、洛阳、建康、成都 | 高法统、高税收、高人口 |
| **粮仓区** | 中原、巴蜀、江东、淮南 | 高粮食产出，养活大军 |
| **边疆军事区** | 幽燕、并州、凉州、辽东 | 高兵源、低税收、高风险 |
| **贸易区** | 江东、岭南、河西、西域 | 高税收、低兵源 |
| **文化区** | 齐鲁、中原 | 高合法性、人才产出 |
| **少数民族区** | 云贵、闽越、交趾 | 高自治、整合难度大 |

## 五、法统体系

### 什么是"法统"？

法统是**统治的正当性基础**。在这个游戏中，法统不是抽象概念，而是直接影响：

- **新占领地区的整合速度**（法统高 → 整合快）
- **人才投奔意愿**（法统高 → 人才来）
- **民变概率**（法统低 → 民变多）
- **继承稳定性**（法统低 → 继承危机）

### 法统的五种来源

| 法统来源 | 说明 | 代表帝皇 |
|----------|------|----------|
| **天命** (mandate) | 天降祥瑞、受命于天 | 秦始皇、刘邦 |
| **功业** (military) | 统一天下、开疆拓土 | 汉武帝、李世民 |
| **血统** (ritual) | 世袭正统、宗法继承 | 赵匡胤、康熙 |
| **制度** (bureaucracy) | 科举选才、文官治理 | 杨坚、朱元璋 |
| **民心** (charisma) | 仁政爱民、百姓归心 | 刘备、刘邦 |

### 法统的历史演变

| 时期 | 核心法统 | 说明 |
|------|----------|------|
| 秦 | 法家功业 | "天下一统"本身就是合法性 |
| 汉 | 天命+德治 | 董仲舒"天人感应"，天降灾异以示褒贬 |
| 魏晋 | 禅让 | "禅代"模式，实质为篡位但形式合法 |
| 唐 | 军功+门阀 | 兼重武功与血统 |
| 宋 | 文治+道统 | 理学强化"正统论"，道统与治统合一 |
| 明 | 驱逐异族+天命 | "驱除胡虏，恢复中华" |
| 清 | 多民族统一 | 满汉蒙藏多元法统 |

### 法统在游戏中如何运作？

1. **每个区域有法统记忆**（legitimacyMemory）：记录该地区历史上的法统来源
2. **帝皇的法统类型**（legitimacyTypes）：决定帝皇的合法性基础
3. **法统匹配度**：帝皇的法统类型与区域法统记忆匹配 → 整合更快
4. **法统冲突**：帝皇的法统类型与区域法统记忆冲突 → 整合更慢、民变更高

> 参考：饶宗颐《中国史学上之正统论》、刘泽华《中国政治思想史》

## 六、核心设计原则

### 1. 没有完美的帝皇

每位帝皇都有明确的"历史代价"：

- 秦始皇：统一极快，但民力透支
- 刘邦：人才凝聚，但功臣坐大
- 李世民：全能，但继承阴影
- 赵匡胤：稳定，但军事受限

**游戏意义**：玩家必须在帝皇的强项和弱点之间做权衡。

### 2. 扩张必须有代价

新占领地区不能立刻贡献税收和兵源：

- 整合度从 0 开始，需要时间提升
- 地方势力会抵抗中央集权
- 文化差异会导致民变风险

**史料依据**：秦始皇统一六国后，楚地"楚虽三户，亡秦必楚"的民怨。

### 3. 天命可以转移

合法性不是固定值，会因为以下原因变化：

- 灾异（旱灾、水灾、地震）→ 合法性下降
- 胜战 → 合法性上升
- 暴政 → 合法性下降
- 仁政 → 合法性上升

**史料依据**：董仲舒"天人感应"理论，天降祥瑞灾异以示褒贬。

### 4. 统一不是终点

统一后仍然面临：

- 土地兼并导致的社会矛盾
- 继承危机导致的内战风险
- 边疆威胁导致的军事压力
- 制度僵化导致的治理效率下降

**史料依据**：每个统一王朝都经历了"开国→盛世→危机→中兴→衰亡"的周期。
